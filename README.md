# The Alchemist

## Introduction

Welcome to **The Alchemist**, a creative crafting and discovery game where you combine basic elements to unlock new ones, expand your collection, and uncover all possible recipes.

Starting from just four fundamental components—**air, fire, earth, and water**—you’ll experiment, mix, and discover up to **30 unique items**. Along the way, you’ll earn coins, unlock recipes, and explore a dynamic workshop environment designed to make experimentation both intuitive and rewarding.

---

## Core Gameplay

At its heart, The Alchemist is about **combining items to discover new ones**.

### Starting Point
- You begin with four basic elements:
  - 🌬️ Air  
  - 🔥 Fire  
  - 🌍 Earth  
  - 💧 Water  

### Crafting System
- Drag items from the shelves onto the **workshop table**
- Combine two elements to create a new one
- If the combination is valid:
  - A **new item is discovered**
  - It appears on the shelves
  - A message describes your discovery
- If invalid:
  - Items return to their original positions

### Inventory
- All discovered items are stored on shelves
- You have **unlimited copies** of each discovered element

---

## Progression & Rewards

### Coins
- Each new discovery rewards you with coins
- More valuable items grant more coins
- Your total coin count is always visible

### Recipe Book
- Access the **Recipe Book** to:
  - View all possible items
  - See which recipes are **locked or unlocked**
- Clicking a recipe shows:
  - Item details
  - Required ingredients (if unlocked)

### Unlocking Recipes
- Some recipes are locked
- You can unlock them by **spending coins**
- If you don’t have enough coins:
  - The action is blocked

---

## User Interface & Flow

### Starting the Game
1. Launch the game
2. Click **“New Game”**
3. Read the instructions overlay
4. Enter the workshop
   
### Workshop Area
- Central table for combining items
- Shelves in the background store discovered elements
- New items appear dynamically as you progress

### Feedback System
- Discovery messages appear when new items are created
- Progress updates inform you of new findings
- Clear error handling for invalid actions

---

## Goal

Your objective is simple:

> **Discover all 30 elements and complete the game**

Once all items are unlocked:
- A **completion message** appears
- You’ve fully mastered the workshop!
